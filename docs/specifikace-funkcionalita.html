Otevřené body — nutné vyjasnit s klientem

Kritické (blokují vývoj)

  1. No-show penalizace — je to 100 Kč nebo 100 % ceny lekce?
  2. Stripe přístupy — čekáme na API klíče od klienta

Důležité (potřebné brzy)

  1. Minimální zůstatek kreditu pro rezervaci — jaká je minimální částka?
  2. Heslo a číslo karty — jak přesně funguje vztah mezi nimi?
  3. Doména — potvrzení booking.zone4you.cz a nastavení DNS
  4. HTTPS certifikát — je na serveru, nebo ho musíme řešit?

Nekritické (lze řešit později)

  1. Reformer odlišnosti — jaké přesně?
  2. WhatsApp Business API — přístup a nastavení
  3. Gamifikace — přesná pravidla pro odznáčky
